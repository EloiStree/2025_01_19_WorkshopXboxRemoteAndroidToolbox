
# Workshop: Xbox Remote Android Toolbox

Following:  https://github.com/EloiStree/2026_01_16_WorkshopNidGamingSharpArcade/blob/main/README.md

## Objectifs de la journée de cours

Deux objectifs ce jour de cours :
* apprendre à utiliser des interfaces 2D Unity pour faire un contrôleur à distance sur Android ;
* s’amuser à travailler en équipe pour relever le défi.


## Contenu du workshop

Prolongement de l’exercice sur les Input Actions, mais appliqué aux interfaces 2D de Unity.
Vous trouverez lié à cet exercice un code qui permet d’envoyer, à une adresse IP avec un port, un index et une valeur à exécuter.

Le concept est simple :
Sur mon écran est affiché un jeu à 4 manettes en coopération. 
Essayez de vous entraîder pour aller le plus loin possible.
* 4 manettes = 4 équipes
* Chaque équipe doit avoir un "senior" et un junior

Comme je vois que la classe est divisée entre ceux qui n’ont jamais touché à la programmation et ceux qui connaissent déjà un langage :
* Le but du junior est de barrer un maximum de mots sur la feuille A4.
* Le but du senior est de se faire tester sur ses connaissances pendant le code de ce challenge ;).
